# Electricity Load Prediction & Recommendation System

## Problem Statement
Predict electricity load and generate smart recommendations using ML and DL models.

## Models Used
- Linear Regression
- KNN
- Decision Tree
- Random Forest
- SVM
- ANN (Deep Learning)
- Naive Bayes (Classification)

## Features
- TimeSeriesSplit
- Model Comparison
- Recommendation System
- Clustering

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run notebook:
   Open main.ipynb and click 'Run All'
3. (Optional App):
   streamlit run app.py
